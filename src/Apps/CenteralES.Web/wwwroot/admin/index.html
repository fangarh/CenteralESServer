Что проверяется

PostgreSQL: возможность открыть соединение и выполнить `select 1`.
Processing schema: наличие обязательных таблиц очереди, результатов, admin и API keys.
Temporary storage: запись, чтение, удаление health-файла и capacity guard.

Граница MVP

Эта страница не вызывает внешний `pdf2txt` и не отправляет PDF.

Безопасный diagnostic probe будет отдельным действием с audit.
